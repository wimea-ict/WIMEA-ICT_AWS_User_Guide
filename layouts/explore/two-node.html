Two meter node

Used to measure the temperature and humidity.

Description

Two meter node is designed in the following ways
The RSS2 Mote is connected first, lithium-ion capacitor is connected on the RSS2 using jumper wires, Boot converter is connected on capacitor and Solar plane is connected on converter to supply power and finally SHT25 sensor is connected on RSS2 using jumper wires. The node is installed 10m from the ground. It was built for WIMEA-ICT Gen 3 Automatic weather stations.

Components

  • RSS2 mote:To collect, process, store and transmit weather parameters to a remote repository.
  • SHT25 sensor:An efficient device to measure the temperature and humidity.
  • lithium-ion capacitor:A hybrid type of capacitor classified as a type of supercapacitor. Activated carbon is typically used as the cathode.
  • Boost converter:A DC-to-DC power converter that steps up voltage (while stepping down current) from its input (supply) to its output (load).
  • 0.5w Solar Plane:To supply power to the RSS2 MOTE.
  • I2C cable:For managing time and date.
  • 12mm pipe:for holding SHT25 Sensor
