10 Meter Node

Used to measure wind direction and wind speed.

Description

The 10m node is designed in the following ways
The RSS2 Mote is connected first, lithium-ion capacitor is connected on the RSS2 using jumper wires, Boot converter is connected on capacitor, Solar plane is connected on converter to supply power, Anenometer is connected on RSS2 using jumper wires and Wind vane sensor is connected on RSS2 using jumper wires. The node is installed 10m from the ground. It was built for WIMEA-ICT Gen 3 Automatic weather stations.

Components

  • RSS2 mote:To collect, process, store and transmit weather parameters to a remote repository.
  • lithium-ion capacitor:A hybrid type of capacitor classified as a type of supercapacitor. Activated carbon is typically used as the cathode.
  • Boot converter:A DC-to-DC power converter that steps up voltage (while stepping down current) from its input (supply) to its output (load).
  • Anenometer:A device used for measuring wind speed and direction.
  • Wind vane:A tool for measuring wind direction.
  • Solar plane:To supply power to the RSS2 MOTE.
