Information About WIMEA-ICT AWS

The WIMEA-ICT Automatic Weather Station (AWS) is composed four wireless sensor nodes. These nodes are named after the distance of placement of the sensors, which are connected to the respective wireless sensor nodes. Three of the nodes are source nodes, which collect weather data and send it using IEEE 802.15.4 protocol to the fourth wireless sensor node, also known as the sink node. The source nodes include the 10m, 2m and ground node. The sink node receives data from the source nodes and stores them temporarily on the gateway, which later transmits it data to a repository, over the Internet. Each node is equipped with a power supply.

Description

The WIMEA-ICT AWS is composed of

  • Ten meter node:Wind Speed, Wind Direction, Humidity, Solar radiation
  • Two meter node:Temperature, Humidity
  • Ground node:Soil Temperature, Soil Moisture
  • Sink node.Temperature, atmospheric pressure, Humidity.
